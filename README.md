介绍
===
这是 MIT6.828 的个人学习笔记，限于非计算机专业出身，可能会有不少纰漏，烦请大神斧正。
个人认为，操作系统是计算机专业课之中综合性很强的一门课，牵涉面广，知识点多，相对也非常难学。

大名鼎鼎的 6.828 的最大的特点就是基于实践。自己写一个操作系统，把坑都踩过一遍，自然就明白了。
